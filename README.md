# 리더보드 대시보드

엑셀 데이터 기반 성적 순위 시스템

## 기능

### 1. 로그인 시스템
- 사번만으로 간편 로그인
- 비밀번호 불필요

### 2. 관리자 기능
- 엑셀 파일 업로드
- 지역단, 사번, 이름, 성적, 차월 데이터 관리

### 3. 대시보드
- **지점별 순위**: 전체 직원 순위
- **지역단별 순위**: 지역단 기준 순위
- **신인 순위**: 차월 12개월 이하 직원 순위

### 4. 개인 맞춤 기능
- 본인 순위가 15등 안에 있으면 중간에 하이라이트 표시
- 본인 순위가 15등 밖이면 15등 바로 아래에 표시
- 개인 통계 (순위, 포인트, 전체 참가자 수)

## 엑셀 파일 형식

엑셀 파일은 다음 열을 포함해야 합니다:

| 지역단 | 사번 | 이름 | 성적 | 차월 |
|--------|------|------|------|------|
| 정동   | 1001 | 김철수 | 2005073 | 24 |
| 불광   | 2001 | 이영희 | 1583626 | 18 |

**CSV 파일을 엑셀로 변환하는 방법:**
1. `sample-data.csv` 파일을 엑셀에서 열기
2. "다른 이름으로 저장" → "Excel 통합 문서 (*.xlsx)" 선택
3. 저장 후 관리자 페이지에서 업로드

## 실행 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 사용 방법

1. **처음 사용시**:
   - 로그인 화면에서 "관리자 페이지" 클릭
   - 엑셀 파일 업로드 (sample-data.csv를 xlsx로 변환하여 사용 가능)

2. **일반 사용자**:
   - 로그인 화면에서 사번 입력
   - 대시보드에서 자신의 순위 확인
   - 지점/지역단/신인 탭을 전환하여 다양한 순위 확인

3. **관리자**:
   - 대시보드 하단의 "관리자 페이지" 버튼 클릭
   - 새로운 엑셀 파일 업로드로 데이터 업데이트

## 기술 스택

- React 18
- TypeScript
- Vite
- Tailwind CSS
- XLSX (엑셀 파싱)
- Lucide React (아이콘)

## 데이터 저장

- 업로드된 데이터는 브라우저 로컬 스토리지에 저장됩니다
- 브라우저를 닫아도 데이터가 유지됩니다
- 새로운 엑셀 파일을 업로드하면 기존 데이터를 덮어씁니다
